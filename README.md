# VRMLviewer
View ancient VRML 1.0 and VRML 2.0 files and convert them into modern GLTB format

## Supported VRML formats:
- 1.0
- 2.0

## Supported file formats:
- 3d meshes:
   - .wrl (plain text)
   - .wrz (zipped .wrl)
   - .zip (zipped .wrl structure)
- Textures:
   - .jpg 
   - .rgb (Silicon Graphics format)

## Demo files

### NASA Pathfinder/Sojourner mission (1997)
- altitude50k - Landing site colored by terrain altitude
